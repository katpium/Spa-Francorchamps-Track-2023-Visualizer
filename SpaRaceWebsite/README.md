# Spa Race Website

A C# ASP.NET Core web project that visualizes the 2023 Spa-Francorchamps Formula 1 race using OpenF1 API location data.

## Features

- Calls the OpenF1 API from a C# backend
- Serves a local website using ASP.NET Core
- Displays a Spa-Francorchamps track image
- Animates a driver dot using real location coordinates
- Uses HTML, CSS, and JavaScript for the frontend

## Tech Stack

- C#
- ASP.NET Core
- JavaScript
- HTML/CSS
- OpenF1 API
